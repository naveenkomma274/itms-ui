# itms-ui
Internal Task Management System UI - Front End Code

## Tech Stack
- React
- Axios
- Context API
- Material UI / Bootstrap

## Features
- Role-based dashboards (Admin / Manager / Employee)
- Task creation & tracking
- Notifications UI
- Employee appreciation module

## Setup
### React Terminal Commands
React app folder creation: npm create vite@latest app_name
Select: React and JavaScript
Install node packages: npm install
Running the app: npm run dev
Command name: npm create vite@latest itms_ui
App component is the base component or root component
Add Bootstrap to React App using NPM: npm install bootstrap --save
npm install react-bootstrap bootstrap
npm list react-router-dom
